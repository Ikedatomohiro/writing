# /plan コマンド

機能実装やリファクタリングの計画を作成します。

## 使用方法

```
/plan [機能の説明]
```

## 実行内容

1. `.claude/agents/planner.md` のガイドラインに従って計画を作成
2. 既存コードベースを分析（Read, Grep, Glob使用）
3. 実装ステップを依存関係順に整理
4. テスト戦略を含む包括的な計画を出力

## 計画に含まれる内容

- 概要
- 要件リスト
- アーキテクチャ変更
- 実装フェーズ（具体的なファイルパスと関数名）
- テスト戦略
- リスクと対策
- 成功基準

## 例

```
/plan ユーザー認証機能を追加
/plan APIレスポンスのキャッシュ機構を実装
/plan データベーススキーマをリファクタリング
```
