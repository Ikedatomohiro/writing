# /architect コマンド

システムアーキテクチャの設計・レビューを行います。

## 使用方法

```
/architect [設計課題の説明]
```

## 実行内容

1. `.claude/agents/architect.md` のガイドラインに従って分析
2. 現状のアーキテクチャを把握
3. 複数のアプローチを比較検討
4. トレードオフを文書化
5. ADR（Architecture Decision Record）を作成

## 対応する課題

- 新機能のシステム設計
- スケーラビリティの改善
- 技術的負債の解消
- パフォーマンスボトルネックの特定

## 例

```
/architect マイクロサービス化の設計
/architect キャッシュ戦略の検討
/architect データベース分割の設計
```
