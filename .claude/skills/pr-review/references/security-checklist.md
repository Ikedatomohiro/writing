# セキュリティチェックリスト

## 認証・認可

| チェック項目 | 確認内容 |
|-------------|---------|
| 認証バイパス | 認証が必要なエンドポイントに認証チェックがあるか |
| 認可チェック | リソースへのアクセス権限を検証しているか |
| セッション管理 | セッションIDは安全に生成・管理されているか |

## 入力検証

| チェック項目 | 確認内容 |
|-------------|---------|
| バリデーション | すべてのユーザー入力を検証しているか |
| 型チェック | 期待する型かどうか確認しているか |
| 境界値 | 数値の範囲、文字列の長さを制限しているか |

## インジェクション対策

| チェック項目 | 確認内容 |
|-------------|---------|
| SQL | パラメータ化クエリを使用しているか |
| XSS | 出力時にエスケープしているか |
| コマンド | シェルコマンドにユーザー入力を渡していないか |
| パス | パストラバーサル攻撃を防いでいるか |

## 機密情報

| チェック項目 | 確認内容 |
|-------------|---------|
| ハードコード | APIキー、パスワードがコード内にないか |
| ログ出力 | 機密情報をログに出力していないか |
| エラーメッセージ | スタックトレースを公開していないか |
| Git履歴 | 過去のコミットに機密情報がないか |

## 依存関係

| チェック項目 | 確認内容 |
|-------------|---------|
| 脆弱性 | 既知の脆弱性を持つライブラリを使用していないか |
| バージョン | セキュリティアップデートが適用されているか |
