# ブログ用ヘッダーコンポーネント 仕様書

## 目的

ブログ公開ページ全体で共通使用するヘッダーナビゲーションを提供し、ユーザーがカテゴリ間を簡単に移動できるようにする。

## 要件

### 1. ロゴ/サイト名表示

- サイト名「Writing」を表示
- クリックでトップページ（/）へ遷移
- テーマカラー（accent）を使用

### 2. カテゴリナビゲーション

| カテゴリ | パス | テーマ |
|---------|------|--------|
| 資産形成 | /asset | investment |
| プログラミング | /tech | programming |
| 健康 | /health | health |

- 各カテゴリへのリンクを表示
- 現在のカテゴリをハイライト表示（accent色）
- ホバー時にアンダーライン表示

### 3. モバイルメニュー（レスポンシブ対応）

- md以下のブレークポイントでハンバーガーメニューを表示
- md以上ではナビゲーションを水平表示
- メニュー開閉時のアニメーション

### 4. スタイル

- 背景色: bg-card（白）
- 下ボーダー: border色
- 高さ: 64px（デスクトップ）、56px（モバイル）
- 最大幅: 1280px（中央寄せ）

## 技術要件

- React Server Componentsは使用しない（"use client"）
- Chakra UIコンポーネントを使用
- CSS変数でテーマカラーに対応
- TypeScriptで型安全に実装
- アクセシビリティ対応（aria属性、キーボードナビゲーション）

## 成功基準

1. すべてのテストがパス
2. 3カテゴリすべてへのナビゲーションが機能
3. 現在のカテゴリがハイライトされる
4. モバイルでハンバーガーメニューが機能
5. ビルドが成功する
