# Pagination コンポーネント仕様書

## 目的・背景

記事一覧ページでページネーションが必要。ユーザーが複数ページにわたる記事リストをナビゲートできるようにする。

GitHub Issue: #22

## 機能要件

### 基本機能

1. **前へ/次へボタン**
   - 前のページへ移動するボタン（chevron-left アイコン）
   - 次のページへ移動するボタン（chevron-right アイコン）
   - 最初のページでは「前へ」を非活性化
   - 最後のページでは「次へ」を非活性化

2. **ページ番号表示**
   - 現在のページ番号を表示
   - クリックで該当ページへ移動

3. **現在ページハイライト**
   - アクティブなページはアクセントカラーで強調表示
   - 非アクティブなページは通常スタイル

4. **省略表示**
   - 総ページ数が多い場合、省略記号（...）で中間ページを省略
   - 例: 1 ... 5 6 7 ... 20

## 非機能要件

### テーマカラー対応

- アクティブページの背景色はカテゴリ別テーマカラーを使用
- CSS変数 `--accent` を使用（カテゴリに応じて切り替え）
- デフォルト値を設定（Investment: #0891B2）

### アクセシビリティ

- `aria-label` でナビゲーション要素であることを明示
- `aria-current="page"` で現在ページを明示
- キーボードナビゲーション対応
- フォーカス時の視覚的インジケータ

## Props インターフェース

```typescript
interface PaginationProps {
  /** 現在のページ番号（1始まり） */
  currentPage: number;
  /** 総ページ数 */
  totalPages: number;
  /** ページ変更時のコールバック */
  onPageChange: (page: number) => void;
  /** 表示するページ番号の最大数（デフォルト: 5） */
  maxVisiblePages?: number;
}
```

## デザイン仕様

### ボタンスタイル

| 状態 | 背景色 | 文字色 | ボーダー |
|------|--------|--------|----------|
| アクティブ | `--accent` | #FFFFFF | なし |
| 通常 | `--bg-card` | `--text-secondary` | `--border` 1px |
| ホバー | `--bg-surface` | `--text-secondary` | `--border` 1px |
| 非活性 | `--bg-card` | `--text-muted` | `--border` 1px |

### サイズ

- ボタンサイズ: 40x40px
- 角丸: 8px
- ギャップ: 8px
- フォントサイズ: 14px
- フォントファミリー: Inter

### アイコン

- 前へ: lucide `chevron-left`
- 次へ: lucide `chevron-right`
- サイズ: 20x20px

## 省略ロジック

```
totalPages <= maxVisiblePages の場合:
  すべてのページを表示

totalPages > maxVisiblePages の場合:
  - 最初のページ (1) は常に表示
  - 最後のページ (totalPages) は常に表示
  - 現在ページの前後1ページを表示
  - 省略が必要な箇所に "..." を表示

例: currentPage=6, totalPages=20, maxVisiblePages=5
  表示: 1 ... 5 6 7 ... 20
```
